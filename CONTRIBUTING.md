# Правила внеску / Contribution Guidelines

## 🇺🇦 Українська

Кожен змістовний коментар, опис Issue, Pull Request і коментар у коді має такий порядок:

1. Український текст.
2. Англійський переклад із тим самим змістом.

Короткі технічні коментарі, які не потребують пояснення, можна не дублювати. Ідентифікатори в коді завжди пишемо англійською.

Робочий процес: Issue → окрема гілка → перевірка → Pull Request.

---

## 🇬🇧 English

Every meaningful comment, Issue description, Pull Request description, and code comment must follow this order:

1. Ukrainian text.
2. An English translation with the same meaning.

Short technical comments that need no explanation do not have to be duplicated. Code identifiers are always written in English.

Workflow: Issue → dedicated branch → verification → Pull Request.
