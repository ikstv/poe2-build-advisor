# Бачення продукту / Product Vision

## 🇺🇦 Українська

PoE 2 Build Advisor допомагає гравцеві обрати практичний білд для його реального прогресу, а не просто показує загальний список популярних білдів.

Перша версія не зчитує дані безпосередньо з гри й не виконує автоматизованих дій у ній. Користувач вводить свої параметри, а програма прозоро ранжує підтримувані білди й показує шлях розвитку.

## 🇬🇧 English

PoE 2 Build Advisor helps a player choose a practical build for their actual progression rather than merely showing a general list of popular builds.

The first version does not read directly from the game and does not perform automated in-game actions. The user enters their parameters, and the application transparently ranks supported builds and shows a progression path.
